# Toy-Project

### 주제
마켓컬리 사이트의 몇가지 기능들 구현해보기
---
언어 : 자바


DB : Mysql


프레임워크 : 스프링부트, 부트스트랩 템플릿
---


#### 개발할 주요 기능 list
1. 검색 기능
2. 로그인
3. 장바구니
4. 마이페이지 (주문내역, 찜한상품, 상품후기, 상품문의)
